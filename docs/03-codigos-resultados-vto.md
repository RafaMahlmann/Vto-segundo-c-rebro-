## Estrutura dos Grupos

```
Grupo A (Codigos 10-23): VTO realizada com interligacao identificada
Grupo B (Codigos 31-35): VTO sem interligacao com rede coletora
Grupo C (Codigos 51-61): VTO nao realizada (impedimentos diversos)
Grupo D (Codigos 71-79): VTO prorrogada (adiamento justificado)
```

## Observacoes para o Aplicativo

1. **Busca por Codigo:** O app deve permitir busca rapida por numero do codigo
2. **Filtro por Grupo:** Agrupamento visual por categoria (A/B/C/D)
3. **Indicador de Aplicacao:** "Nova" vs "Existente" define se o codigo aplica para ligacoes novas ou existentes
4. **Referencias Cruzadas:** Varios codigos citam normas (IA/COM/0453, IA/AMB/0033, ABNT 8160/99) — o app pode vincular essas referencias
5. **Status do Usuario:** Muitos codigos envolvem "usuario orientado e notificado" — o app deve registrar essa comunicacao
